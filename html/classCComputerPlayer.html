Detailed Description

This class defines the API to place the move of the computer player on the board.

Note
The computer decides to attack , defend or block fork as per the Opponents move. Computer may never lose.

void CComputerPlayer::findBestMove ( BoardState_t  tileinfo)
private

API to decide the best possible row and column based on the current state of the board.

decides best move by checking the current state of board in the order :
- empty board
- computer can win in next move
- needs to stop opponent from winning
- block opponent fork
- empty corner
- empty side

Parameters
tileinfoTile that the Computer player is going to place on the board.
